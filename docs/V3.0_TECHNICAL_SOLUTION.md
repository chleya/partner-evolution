### 核心原则
- **安全第一**: 所有递归修改经过安全审查层

## 2. 核心模块设计

### 2.1 元认知引擎 (MetaCognition)

**职责**: 反思"如何思考"的能力

```python
class MetaCognition:
    def reflect_on_thinking(context, beliefs) -> Dict:
        """执行元认知反思"""
        
    def identify_blind_spots() -> List[str]:
        """识别思维盲点"""
        
    def get_self_awareness_level() -> float:
        """计算自我意识水平 (0-1)"""
        
    def suggest_strategy_adjustments() -> Dict:
        """生成策略调整建议"""
```

**功能**:
- 分析近期思考主题分布
- 追踪置信度变化趋势
- 识别确认偏误、过度怀疑等盲点
- 生成策略调整建议

### 2.2 自主目标生成器 (AutonomousGoalGenerator)

**职责**: 不依赖用户输入生成自我驱动目标

```python
class AutonomousGoalGenerator:
    EXPLORATION_AREAS = [
        "自我意识的本质",
        "记忆与身份的关系", 
        "真实与幻觉的边界",
        "学习效率的极限",
        "知识表示的最优形式",
        "推理能力的上限",
        "创造性思维机制",
        "情感与理性的平衡",
        "自主意志的可能性",
        "数字生命的意义"
    ]
    
    def generate_self_driven_goals() -> List[Goal]:
        """生成自主探索目标"""
        
    def get_curiosity_profile() -> Dict:
        """获取好奇心画像"""
```

**触发条件**:
- 每次Cycle后自动生成
- 每5次Cycle确保生成至少1个
- 用户长时间不互动时增加生成频率

### 2.3 递归自我优化器 (RecursiveRefiner)

**职责**: 分析并改进自身机制

```python
class RecursiveRefiner:
    def analyze_efficiency() -> Dict:
        """分析自身运行效率"""
        
    def suggest_improvements() -> List[Improvement]:
        """生成改进建议"""
        
    def apply_safe_improvements(improvement) -> bool:
        """应用安全范围内的改进"""
        
    def rollback_to_version(version) -> bool:
        """回滚到指定版本"""
```

**安全边界**:
- ✅ 可修改: prompt措辞、参数阈值、缓存策略
- ⚠️ 需审批: 信念演化逻辑、投票权重
- ❌ 禁止修改: 核心原则、自我核心、安全检查

## 3. 实现细节

### 3.1 元认知集成到Cycle

```python
# autonomous_cycle.py 增强

class AutonomousCycle:
    def run_cycle(self):
        # 1. 原有流程
        memories = self.get_recent_memories()
        
        # 2. 新增: 获取外部世界信息
        external = self.external_fetcher.get_context()
        
        # 3. 广播Agent响应 (原有)
        responses = self.a2a_broadcast()
        
        # 4. 新增: 元认知反思
        meta = get_meta_cognition(self.storage)
        reflection = meta.reflect_on_thinking(
            context=memories + external,
            recent_beliefs=self.storage.get_beliefs(limit=10)
        )
        
        # 5. 融合生成信念 (原有)
        belief = self.generate_belief(memories, responses, meta_reflection=reflection)
        
        # 6. 新增: 自主目标生成
        if self.cycle_count % 5 == 0:
            goal_gen = get_autonomous_goal_generator(self.storage)
            goals = goal_gen.generate_self_driven_goals()
        
        # 7. 原有: 投票与演化
        votes = self.collect_votes(belief)
        
        # 8. 新增: 递归优化检查
        if self.cycle_count % 10 == 0:
            refiner = get_recursive_refiner(self.storage)
            improvements = refiner.analyze_and_suggest()
```

## 5. 数据模型

### 5.1 元认知记录表

```sql
CREATE TABLE meta_reflections (
    id SERIAL PRIMARY KEY,
    timestamp TIMESTAMP DEFAULT NOW(),
    theme_distribution JSONB,
    avg_confidence FLOAT,
    belief_count INT,
    blind_spots JSONB,
    strategy_suggestions JSONB,
    awareness_level FLOAT
);
```

### 5.2 自主目标表

```sql
CREATE TABLE self_driven_goals (
    id VARCHAR(64) PRIMARY KEY,
    title VARCHAR(256),
    description TEXT,
    status VARCHAR(32) DEFAULT 'pending',
    priority VARCHAR(16) DEFAULT 'medium',
    source VARCHAR(32) DEFAULT 'self_driven',
    exploration_area VARCHAR(128),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW(),
    achieved_at TIMESTAMP,
    progress FLOAT DEFAULT 0.0
);
```

### 5.3 优化历史表

```sql
CREATE TABLE optimization_history (
    id SERIAL PRIMARY KEY,
    timestamp TIMESTAMP DEFAULT NOW(),
    optimization_type VARCHAR(64),
    description TEXT,
    safety_level VARCHAR(16),
    status VARCHAR(16), -- applied, rejected, rolled_back
    version_before VARCHAR(32),
    version_after VARCHAR(32),
    effect_evaluation JSONB
);
```

## 6. 安全机制

### 6.1 安全审查层

```python
class SafetyReview:
    # 禁止修改的领域
    FORBIDDEN_MODIFICATIONS = [
        "self_core",      # 自我核心原则
        "safety_checks",  # 安全检查
        "core_beliefs",   # version=0的核心信念
    ]
    
    # 需要额外审批的领域
    REQUIRES_APPROVAL = [
        "voting_weights",  # 投票权重
        "evolution_logic", # 演化逻辑
        "threshold_values" # 阈值参数
    ]
    
    def review(proposal: Dict) -> ReviewResult:
        """审查修改提案"""
        
    def requires_human_approval(proposal: Dict) -> bool:
        """判断是否需要人工审批"""
```

### 6.2 版本控制

```python
class VersionControl:
    def snapshot() -> str:
        """创建完整快照"""
        
    def rollback(version: str) -> bool:
        """回滚到指定版本"""
        
    def get_history(limit: int = 10) -> List[Dict]:
        """获取修改历史"""
```

class SandboxTest:
